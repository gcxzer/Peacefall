# role_set_engines

`role_set_engines` 放具体版型的游戏流程实现。每个版型建议一个独立 Python 模块。

## 当前版型

- `six_player_classic.py`：6 人经典版型，包含 2 狼人、2 平民、1 预言家、1 女巫。

## 新增版型建议

新增版型时，优先创建新的版型模块，例如：

```text
src/role_set_engines/nine_player_standard.py
```

版型模块里应该包含：

- 该版型的 `run_game` 入口。
- 夜晚和白天阶段推进。
- 该版型独有的角色行动。
- 该版型独有的胜负检查。

跨版型可复用的逻辑应抽到 `src/engine/`，不要在多个版型模块里复制。

## 6 人经典版型

`six_player_classic.py` 实现当前默认版型，流程是：

1. 法官宣布游戏开始。
2. 夜晚阶段：
   - 狼人私聊并由主刀狼提交击杀目标。
   - 预言家查验一名玩家。
   - 女巫决定是否使用解药或毒药。
   - 结算夜晚死亡。
3. 白天阶段：
   - 工具随机指定发言起点。
   - 存活玩家依次发言。
   - 存活玩家投票放逐。
   - 结算放逐。
4. 检查胜利条件。
