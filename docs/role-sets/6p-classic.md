# 6 人经典版型

英文代号：`6P Classic`

## 角色配置

| 阵营 | 角色 | 数量 |
| --- | --- | ---: |
| 狼人阵营 | 狼人 | 2 |
| 好人阵营 | 平民 | 2 |
| 好人阵营 | 预言家 | 1 |
| 好人阵营 | 女巫 | 1 |

总人数：6

## 胜利条件

- 好人阵营胜利：所有狼人出局。
- 狼人阵营胜利：达成屠边条件，即所有平民出局，或所有神职出局。

## 夜晚流程

1. 狼人共同选择一名玩家击杀。
2. 预言家查验一名玩家的阵营。
3. 女巫选择是否使用解药或毒药。

## 白天流程

1. 公布夜晚死亡信息。
2. 所有存活玩家按顺序发言。
3. 所有存活玩家投票放逐一名玩家。
4. 结算放逐结果并检查胜利条件。

## 设计说明

这是推荐优先实现的 MVP 版型，因为它覆盖了狼人杀最核心的循环：

- 狼人的隐藏身份和伪装发言。
- 预言家的查验信息和真假对跳。
- 女巫的夜晚干预能力。
- 平民只能依靠发言、推理和投票。
